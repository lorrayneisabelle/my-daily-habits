# 🌐 Projeto My Daily Habits Curso FullStack - Módulo 4 - Introdução ao React | ITEAM - RR.

## 🚀 Sobre o projeto

O **My Daily Habits** é uma aplicação web desenvolvida utilizando **React** no curso FullStack - Módulo 4 do ITEAM - RR, com foco no gerenciamento de hábitos diários e na aplicação de boas práticas de desenvolvimento front-end.

## 🧠 Estrutura do projeto

O projeto segue uma arquitetura comum em aplicações React modernas, incluindo:

- **Pages (Páginas):** responsáveis por representar as telas da aplicação  
- **Components (Componentes):** elementos reutilizáveis da interface  
- **Layout:** estrutura base que define o padrão visual (header, sidebar, etc.)  
- **Rotas:** navegação entre páginas utilizando roteamento  
- **Organização modular:** separação de responsabilidades para facilitar manutenção  

## ⚙️ Tecnologias utilizadas

- React  
- JavaScript 
- CSS  

## 🎯 Objetivo

O objetivo do My Daily Habits é praticar conceitos essenciais do React, como componentização, reutilização de código e organização de projetos, simulando um ambiente real de desenvolvimento.
